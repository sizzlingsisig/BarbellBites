# BarbellBites Frontend

React + TypeScript + Vite client for BarbellBites.

## Run Locally

```bash
npm install
npm run dev
```

Default URL: `http://localhost:5173`

## Scripts

- `npm run dev`: start Vite dev server
- `npm run build`: type-check and build production assets
- `npm run preview`: preview production build
- `npm run lint`: run ESLint

## Stack

- React 19
- React Router
- TanStack React Query
- Zustand
- Axios
- Mantine UI
- Zod + React Hook Form
- Sonner notifications

## API Integration

- Backend API expected at `http://localhost:3000`
- API modules are under `src/api/`
- Routes/pages are under `src/router/`, `src/pages/`, and `src/layouts/`
